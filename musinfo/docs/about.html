A real-time audio analysis system that extracts musical structure from live instrument input and drives generative visuals through TouchDesigner.

01 — What is it

Sound made visible

MUSINFO listens to a live performance, analyses pitch, genre, and harmonic content in real time, and translates that information into visual parameters.

The result is a performance system where the visuals are not designed in advance — they emerge from the music itself, frame by frame.

03 — Stack
Frontend
Tauri + React
Rust backend, React UI. Native desktop app with a web frontend.
Analysis — Windows
Python + aubio
Real-time pitch detection via YIN algorithm. Hz-to-note conversion with confidence filtering.
Analysis — WSL
Essentia + TF
Discogs-EffNet genre classification. 400-label model with 4s sliding window at 16kHz.
Audio routing
VB-Cable + Reaper
16-channel WDM-KS virtual cable for multi-instrument routing without ASIO conflicts.
Visuals
TouchDesigner
OSC In DAT receives analysis results. Real-time generative visual output driven by musical data.
Protocol
OSC + TCP
OSC for Tauri↔TouchDesigner. TCP for Windows→WSL audio chunk streaming.
